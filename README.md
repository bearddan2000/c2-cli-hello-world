# c2-cli-hello-world

## Description
A POC for c2 programming language.

## Tech stack
- g++
- make
- ocaml

## Docker stack
- ubuntu:22.04
- rzuckerm/c2

## Build note
This will take 1+ hours to build

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
